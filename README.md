# 📊 U.S. Sales Performance and Profitability Insights Dashboard

## 📌 Project Overview
This project analyzes **U.S. sales performance and profitability** using cleaned sales data.  
The analysis is performed in **Excel** and visualized in **Power BI**, providing interactive insights into sales trends, profits, and customer distribution.

The goal is to help businesses understand sales patterns, identify top-performing products and regions, and track profitability over time.

---

## 🛠️ Tools Used
- **Microsoft Excel** – Data cleaning and preparation  
- **Power BI** – Dashboard creation and visualization

## 🔍 Data Analysis Process
Data Cleaning in Excel
Removed duplicates
Handled missing values
Standardized product and category names
Data Modeling in Power BI
Linked tables and calculated measures
Dashboard Design
Created interactive visuals: bar charts, pie charts, waterfall charts
Added slicers for City, Product, Month, Year
Insight Generation
Analyzed sales trends, top products, and profitability
Identified high-performing regions and customer segments
